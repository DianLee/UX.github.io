About the project

從「善用每一度電」的願景出發，打造全台唯一用 LINE 就可看見用電的全新電力管理服務，讓用電變得簡單有感！

隨著社會與產業發展，用電量或是能源使用量不斷向上攀升，然而臺灣是一個島嶼，除了部分再生能源之外，絕大部分的能源都需要進口原料。原本做為基載的核能發電，原料體積小發電時又不會產生汙染物，非常適合臺灣使用，無奈政府錯誤的能源政策下，大量不穩定的再生能源下就要依靠大量的火力來做為發電主力，在面對減碳排或是零碳排全球潮流下，根本無法達成，更可能造成缺電危機。在開發新的電力之外，廣大的使用者若能節省電力，善用每一度電，也能降低一些缺電危機。

NextDrive 與台灣電力公司共同宣布推出新一代家庭能源管理服務「1%能源行動家」；用戶藉由智慧電表與LINE串接，在手機中即可「看見用電」，如同電力記帳本，還可以設定電費預算與超支提醒。因應能源轉型趨勢，未來還可以整合智慧家電、小型儲能系統、甚至電動車充電樁，成為台灣家戶型需量管理的關鍵角色。

挑戰與目標:

在僅剩四個月的時間內，如何推動整個產品的規劃與內部執行，並與相關團隊(設計、開發、QA、外部廠商)協作，將產品「Line智能電力小管家」，從零至一如期推上線。

DESIGN PROCESS

PROJECT HIGHLIGHT

01. Stakeholder Interview・利益關係人需求訪談

為了釐清事業體的策略、目標以及該產品目前技術支援程度、商業與業務預期成效、指標，並整理出可執行的專案建議與先後順序；訪談了行銷長、該事業體的策略經理以及內部技術顧問。

02. Envision Workshop・品牌服務定位與調性工作坊

「品牌」是公司與顧客溝通的核心，如何有效的與目標族群溝通，提供相對應的需求與服務，並與公司的策略方向符合一致性是非常重要的；故，為了找出產品的服務定位、核心價值以及與關鍵客群的互動調性，透過工作坊的形式集結公司相關部門的利益關係人(執行長、技術長、行銷團隊、技術團隊、業務單位、設計單位)，讓大家互相合作、碰撞想法，最終產生大家都共識的產品服務定位與顧客溝通調性。

03. Project Execution・專案執行過程

在上線時間極為急迫，公司尚未有充裕的內部資源的狀況下，透過以往的經驗協助團隊增加專案推進效率，並在專案執行期間推動與負責下列項目：
- 推動Scrum，擔任Designer並兼任PO角色
進入專案後，因具體需求與技術都充滿不確定性，且上線時間又緊迫的情況下，毅然決然與團隊討論後推動Scrum的開發方式協助專案進行。經團隊討論後，在此專案中擔任Designer與Product Owner角色，以用戶的角度規劃產品功能，與技術單位討論各功能耗費的人力資源後，從商業的價值與技術可行性上擬定開發順序，以及訂定專案主要里程碑與時程，並擔當對內對外，跨部門上下溝通的主要角色。
- 設計管理與Review
在須執行主要的 UX/UI 設計與內外溝通外，也分派設計工作項目，控管設計設計品質與時程，並帶領一位UX設計師、一位UI設計師一起進行專案。
- 協助產品推廣
為了提升產品知名度，在產品上線後，與行銷團隊一起進行一系列的推廣活動（網紅分享、媒體面談、產品發表記者會），並擔任主要對外解說產品的角色。

Highlight Features・重要特色

01.

Feature・適時恰當的提醒

在對的時間點主動與用戶溝通，能維持用戶與品牌的關係，是很多品牌常用的方法；透過 LINE 我們更容易達到這件事情，藉由用戶每段旅程的情境與狀況，找到主動與用戶溝通的接觸點（如：節能計畫提醒、每期電費結算、綁定電表、不同季節省電新知...等），都能讓用戶持續與我們保持互動，堆疊用戶對我們的關係，來延長整個品牌的CLV。

02.

Feature・親切吸引人的調性

透過品牌調性工作坊，我們確立了品牌的溝通調性以及主要角色的性格，並以此規範與目標用戶進行不同脈絡下的互動（如：用電預算爆表提醒、新知的傳播、活動進度的告知、目標達成時的鼓勵...等），降低用戶的操作壓力，並提升用戶的互動頻次。

03.

Feature・用電狀況時時掌握

用戶可主動查詢目前即時用電狀況，且可自行查詢不同的月、週、日時間區間，以及能切換「電費模式」與「用電量模式」兩種；且隨著用電費用的增加，系統會有不同的提醒，讓用戶注意不要超過自己設置的用電預算。

04.

Feature・有脈絡的深層互動

用戶可主動查詢目前即時用電狀況，且可自行查詢不同的月、週、日時間區間，以及能切換「電費模式」與「用電量模式」兩種；且隨著用電費用的增加，系統會有不同的提醒，讓用戶注意不要超過自己設置的用電預算。
